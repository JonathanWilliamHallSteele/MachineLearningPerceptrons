To run the program, navigate to solution.java, compile, then run its main method.

Please feel free to change the inputs and outputs variables, declared at the start of solution.java.
They can be any length, as long as they have an equal number of rows.

Sensor holds weights and activations for each sensor node. The first sensor in sensors[] holds the output node
Instance holds all information for a given row of inputs and outputs
Instances organizes all Instance variables

Forward_Propagation is used to calculate the output
Train_Test is used to adjust weights and bias for one training Instance
